# ESP32 Drone
## Project Description
Charles P. Allen Robotics club's project in 2025-2026 school year is to build a mini drone. Its functions include PID balancing with MPU6050 and basic controls. Future implementations could be FPV camera and automatic landing to a RC boat.

# Components Used
## Controller
- XIAO ESP32C3 with External Antenna
- Perfboard
- 2X Joysticks

## Drone
- XIAO ESP32S3 Sense with External Antenna
- 2X DRV8833
- MPU6050
- 4X 716 Corless Motors
- 3D Printed Drone Frame
